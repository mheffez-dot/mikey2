CPS Vendor Validator

What It Does

You give it a list of vendor websites, and it tells you which ones sell products CPS can write extended warranties on.

🔍

How It Works

  • Fetches each website (direct fetch first, Bright Data for bot-protected sites)
  • Reads page content — title, description, product/category menu
  • Classifies with DeepSeek against CPS coverage policy
  • Covered: electronics, appliances, tools, firearms
  • Excluded: apparel, food, services, software
📊

What You Get

  • Verdict: Pass / Fail / Mixed per domain
  • Confidence score & matched category
  • Real quoted evidence + one-line reason
  • Results written to Google Sheet
  • Email with CSV attached
  • Automatic duplicate skipping
⚡

Fast & Affordable

  • Processes ~20 domains in parallel
  • ~$0.002 per site (often less)
  • Easy sites skip Bright Data proxy
  • Automatic provider outage detection
  • Turns hours of manual work into minutes
